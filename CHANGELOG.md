# Changelog

All notable changes to the `magical-gazelle` CLI and plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.7.0] - 2026-02-16

### Changed

- Forked from [compound-engineering-plugin](https://github.com/EveryInc/compound-engineering-plugin)
- Renamed plugin to magical-gazelle
- Removed Rails/Ruby components, customized for Python/TypeScript workflow
- Added simplicity and elegance standards to Python and TypeScript reviewers
- Added Philosophy section to Python CLAUDE.md template
- Cleaned up all commands for Python/TS stack (agent-browser, localhost:8000, uv/ruff/pytest)
- Removed cursor, droid, gemini CLI targets (keeping opencode, codex, pi)
- Renamed CLI from @every-env/compound-plugin to magical-gazelle
- Updated GitHub source URL to grbellar/magical-gazelle
